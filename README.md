# Project Title

Meeting Scheduler

## Description

Sample UI app made for code assignment. UI is only compatible for web and is not tested on other environments.

## Understanding Structure

Provider structure is used for state management.
Seperate folders are created for Widgets, Pages, Utils, Providers, Data Models

## UI Elements

There is one BookingPage contains Five different widgets listed below:

* TopBarWidget. i.e. for top bar
* DurationWidget (First Section)
* DateWidget (Second Section)
* TimeWidget (Third Section)
* ConfirmationBookingCardWidget (Confirm booking overlay)


## Executing project

Flutter environment is required to run the project.
